# EXT2

这是一个简单的对EXT2文件系统的模仿实现
主要实现了以下功能

+ 文件创建 / 删除接口命令 touch / rm

+ 目录创建 / 删除接口命令 mkdir / rmdir

+ 显示目录内容命令 ls

+ 切换目录命令 cd

+ 读 / 写文件命令 read / write

+ 更改文件权限命令 chmod
